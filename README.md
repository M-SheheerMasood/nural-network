Python - pytorch - numpy - pandas - MNIST dataset - FCNN
This is a full-from-scratch, fully connected neural network with 2 internal layers and a multi-node outer layer. Trained on the numbers dataset to intelligently recognize numbers 0 to 9.
I have also modified this for a smaller scale to explore. The bonus folder has a version where the one who runs the file uploads the image and the label to train the model.
Its state is then saved and can be used to try to guess at any time.
